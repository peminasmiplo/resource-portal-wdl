# resource-portal-wdl

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面的集中存储与分发点。

## 项目简介

`resource-portal-wdl` 旨在为多个独立的 HTML 页面提供统一的归档与发布环境。这些页面可包含工具、信息展示、交互原型等各类内容，彼此间无强关联。仓库本身不提供搜索、导航或内容聚合功能，仅托管原始页面文件。

## 目录说明

```
resource-portal-wdl/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
├── assets/             # 公共资源（如 CSS、JS、图片等）
│   ├── styles/
│   ├── scripts/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要页面存放目录，每个文件为一个独立 HTML 页面。
- `assets/`：存放页面间共享的样式、脚本、图片等资源，避免重复引用。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可直接通过浏览器打开使用。
- 页面之间不共享状态或数据，彼此完全隔离。
- 归档的页面可能随时间更新或移除，不保证长期可用性。
- 页面的具体内容、功能、来源等，请参见各文件内部的注释或说明。

## 维护说明

- 本仓库由维护者不定期更新，不提供固定的发布计划。
- 提交新页面或更新现有页面时，请确保文件结构清晰，资源引用正确。
- 不建议对仓库内容进行外部链接或依赖，以免因页面变动导致失效。
- 如有疑问或建议，可通过 GitHub Issues 或 Pull Requests 进行沟通。

## 许可

本仓库内容按原样提供，不附带任何明示或暗示的保证。具体页面可能包含独立的许可声明，请以页面自身标注为准。
